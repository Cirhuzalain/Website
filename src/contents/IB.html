Inductive Bias for Seq2Seq Model¶

Inductive bias are quintessential for various model architecture, they can be use to add additional information to the model. In this post we'll see how to update the formulation of Machine Translation's objective function with syntactic information during encoding or via multi-task learning[1]. Additionally we'll see how to inject these prior with templating for an image caption model.
